Simple Gulp with SASS template

This is a very simple document startup template. We have gulp with sass set up. We also have autoprefixer and browserSync installed. There are no unnecessary packages and no meddling with your JS or images.

Usage

The usage is fairly simple. `npm install` and later `gulp watch`.
You'll get the hang of it. ;)

SASS Compiling Rules

All compiled files will compile within the app/css directory, under the same filename. For example the default files will compile as app/css/style.css and app/css/print.css.

JS Compiling Rules

All js files are split into two directories: domain - which contains your scripts, and vendor, which contains vendor scripts.

Terms that apply

  1. - All other files including JS and HTML are free to stay in the app folder.
  2. - Generally, you will upload only the contents of the "app" folder to the server.
  3. - Autoprefixer adds browser prefixes only after compiling success of SCSS.
  4. - .gitignore is set up for OSX, Windows and Node.
  5. - On every save of every file, browser will be refreshed.
  6. - There are limits as we watch only for HTML, JS and SCSS files for reload.
  7. - Manual page reload is needed for images directories and other.
